# Biblioteca Interativa
## Funcionalidades
- Ambiente 3D navegável com W-A-S-D e rato
- Biblioteca virtual com diferentes secções
- Livros interativos com painel informativo
- Paginação de conteúdos
- Sons ambientes, passos, porta e interação com livros
- Interface pensada para usabilidade em VR/Web

## Tecnologias
- HTML
- JavaScript
- A-Frame
